# Swarm Settings

Use `SWARM-ORCHESTRATION.prime-mermaid.md` as the external swarm config contract.

Override order:
1. `STILLWATER_SWARM_SETTINGS_FILE`
2. `swarm_settings_file` in `cli/kernel_config.yaml`
3. `cli/extensions/settings/SWARM-ORCHESTRATION.prime-mermaid.md` (if present)
4. `cli/settings/SWARM-ORCHESTRATION.prime-mermaid.md`
