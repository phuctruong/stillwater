# Stillwater: Blank Canvas

This CLAUDE.md is intentionally blank. All context and instructions are injected via swarms through the LLM Portal.

## Architecture

The LLM Portal loads swarms on demand. Each swarm specifies:
- Skill pack (prime-safety + domain skills)
- Persona (optional)
- Model preference
- Rung level
- Expected artifacts

No CLAUDE.md overhead. Pure swarm-driven context injection.
